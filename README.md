# Netflix Power BI Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using the Netflix Movies and TV Shows dataset. The dashboard provides a comprehensive overview of Netflix's content library and helps users explore trends related to content type, ratings, genres, countries, and release years.

The goal of this project is to transform raw data into meaningful visual insights through effective data visualization and dashboard design.

## Dashboard Highlights

The dashboard includes the following key visualizations:

* Total Number of Netflix Titles
* Top 10 Countries by Movies and TV Shows
* Genre Distribution by Total Titles
* Movies vs TV Shows Comparison
* Content Ratings Distribution
* Release Year Trend Analysis

## Key Insights

* Movies represent a larger portion of Netflix's content catalog compared to TV Shows.
* The United States contributes the highest number of titles in the dataset.
* Drama and International genres are among the most frequently available content categories.
* TV-MA and TV-14 are the most common content ratings.
* The number of titles released on Netflix has grown significantly over recent years.
